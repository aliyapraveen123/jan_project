## Troubleshooting

### "No API key found"
- Check `.env` file exists
- Verify `GOOGLE_API_KEY=your_key_here`

### "Could not download audio"
- Video might be age-restricted
- Video might be private
- Try a different video

### "FFmpeg not found"
- Install: `sudo apt-get install ffmpeg`

### Slow performance
- Normal for audio extraction (10-30 seconds)
- Use videos with captions for faster results
